# Online-Buss-Pass-Generation-And-Renewal
Online Bus Pass Generation &amp; Renewal Management System is a web-based application built with Python (Django), MySQL, HTML, CSS, and JavaScript. It allows users to apply and renew bus passes online, download digital ID cards with QR codes, and enables conductors to verify passes via camera scanning, while admins manage routes, users, and approvals.
